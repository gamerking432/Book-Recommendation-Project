📚 Content-Based Book Recommender

A simple and efficient system that recommends books based on their content, not user behavior.

Built using TF-IDF and cosine similarity, it finds books that are actually similar in meaning — then ranks them to give the best results.

⚡ Key Idea
Understand the content of books
Measure similarity mathematically
Recommend the most relevant matches
🛠️ Tech

Python • Pandas • Scikit-Learn
